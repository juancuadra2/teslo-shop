## Installation

1. Clone the repository
    ```bash
    git clone ...
    ```

2. Install dependencies
    ```bash
    npm install
    ```

3. Install nest CLI
    ```bash
    npm install -g @nestjs/cli
    ```

4. Run database
    ```bash
    docker-compose up -d
    ```
5. Clone the ```.env.template``` file and rename it as ```.env```
6. Assigns the environment variables defined in the ```.env``` file

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
